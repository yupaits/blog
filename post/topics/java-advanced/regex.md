# 正则表达式

## 常用正则表达式

|规则|表达式|
|---|---|
|一个或多个汉字|^[\u0391-\uFFE5]+$|
|邮政编码|^[1-9]\d{5}$|
|QQ号码|^[1-9]\d{4,10}$|
|用户名（字母开头 + 数字/字母/下划线）|^[A-Za-z][A-Za-z1-9_-]+$|
|手机号码|^1([358][0-9]\|4[579]\|66\|7[0135678]\|9[89])[0-9]{8}$|
|URL|^((http\|https)://)?([\w-]+\\.)+[\w-]+(/[\w-./?%&=]*)?$|
|18位身份证号|^(\d{6})(18\|19\|20)?(\d{2})([01]\d)([0123]\d)(\d{3})(\d\|X\|x)?$|
|邮箱|^[a-zA-Z_]{1,}[0-9]{0,}@(([a-zA-z0-9]-*){1,}\\.){1,3}[a-zA-z\\-]{1,}$|
