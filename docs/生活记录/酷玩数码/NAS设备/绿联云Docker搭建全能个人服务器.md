# 绿联云Docker搭建全能个人服务器

笔者在2025年将极空间Z2S升级更换成绿联DXP4800，之后就一直将绿联NAS作为主力设备使用。这台NAS设备拥有N100处理器和16G内存，性能十分强劲，用来搭建个人服务器是绰绰有余。

![sun-panel面板展示](/images/生活记录/酷玩数码/NAS设备/绿联云Docker搭建全能个人服务器/绿联nas硬件配置.webp)

经过了一年的练习期，在不停尝试搭配使用各种先进、稳定的工具和服务之后，我终于实现了搭建全能个人服务器的目标。

首先列出个人服务器的核心需求，相信这些也是广大开发者的核心诉求：

| 需求        | 概述                  | 备选项                                                      | 选型依据                                                                                                   |
|-----------|---------------------|----------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| 局域网DNS服务  | 解析域名到内网IP           | • DNSmasq<br>• CoreDNS ✔                                 | CoreDNS是CNCF毕业项目，当前已是云原生标准；其采用单一进程、插件化架构，无已知的安全漏洞历史，十分可靠                                               |
| Web服务器    | 反向代理，HTTPS搭建，静态网站搭建 | • Lucky<br>• nginx<br>• Nginx Proxy Manager<br>• Caddy ✔ | Caddy能自动申请与续期Let's Encrypt证书，并且默认启用；Caddyfile的语法设计得非常直观，配置极其简单；原生支持HTTP/3和QUIC现代化协议；配置重载过程更加平滑，支持零停机更新 |
| 关系型数据库    | 提供数据存储服务            | • MySQL<br>• MariaDB<br>• PostgreSQL ✔                   | PostgreSQL是一个功能更完整、标准兼容性更好、对数据一致性要求更严格的“先进”数据库，在大多数现代Web应用开发中正成为越来越多开发者的首选                             |
| Git代码托管服务 | 管理个人项目代码            | • Gitea ✔<br>• GitLab<br>• Gogs<br>• OneDev              | Gitea资源占用低、代码协作管理功能完善，支持OAuth2/OpenID等认证集成，部署维护备份简单，扩展能力强                                              |
| CI/CD服务   | 项目流水线构建及部署          | • Drone CI ✔<br>• Gitea Actions                          | Drone可独立部署升级，容器化流水线隔离性更强，提供更成熟灵活的插件机制应对复杂构建部署场景                                                        |
| 对象存储服务    | 应用内文件上传下载、对象存储服务    | • MinIO<br>• Garage ✔<br>• SeaweedFS                     | Garage是一个自包含的、去中心化的系统，没有外部依赖，部署和管理都更加简单和直接，兼容S3                                                        |
| 分布式缓存     | 分布式缓存服务             | • Redis ✔<br>• Redis Stack<br>• Valkey                   | Redis是一个高性能、稳定、低延迟的内存数据存储，Redis 8版本开始重新回归开源，目前完全能够满足分布式缓存的需求                                           |
| 单点登录服务    | 打通账号体系，认证授权平台       | • Casdoor<br>• Logto ✔                                   | Logto提供了更现代化的开发者体验、更完善的企业级功能和更活跃的社区生态，拥有开箱即用、美观且可定制的登录界面                                               |
| 配置中心      | 应用配置统一管理、配置动态更新     | • Nacos<br>• R-NACOS ✔                                   | R-NACOS是一个用Rust重新实现的Nacos替代品，它在提供几乎相同核心功能的同时，显著降低了资源消耗，并带来了更高的性能和启动速度                                  |

将以上核心需求对应选型的工具或服务搭建完成之后，日常的开发工作即刻变得高效顺畅。

上述核心需求对应的服务部署绿联NAS Docker的`dns`和`dev`项目，在`life`项目中还有以下个人严选可供挑选：

| 名称          | Docker镜像                            | 概述                                          |
|-------------|-------------------------------------|---------------------------------------------|
| Sun-Panel   | hslr/sun-panel:latest               | 轻量的服务器导航面板，用于将常用的服务链接和工具整合到一个简洁的页面中         |
| Aria2       | superng6/aria2:webui-latest         | 轻量级、多协议的命令行下载工具，支持 HTTP/HTTPS、FTP、BT 和磁力链接等 |
| Drawnix     | pubuzhixing/drawnix:latest          | 开源的在线绘图与白板协作工具，可用于创建手绘风格的图表和草图              |
| Vaultwarden | vaultwarden/server:latest           | 用 Rust 编写的 Bitwarden 密码管理服务非官方替代实现，支持自托管    |
| TaskTrove   | ghcr.io/dohsimpson/tasktrove:latest | 基于 GTD 理念的任务管理和待办事项应用，适合个人高效管理日常事务          |
| Memos       | neosmemo/memos:stable               | 开源、轻量的知识管理与碎片笔记记录工具，用于快速记录和整理信息             |
