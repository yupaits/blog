# unicode

在Go语言开发中，字符串截取是高频操作，尤其涉及中英文混合处理时需特别注意编码问题。本文系统梳理Go语言中4种主流字符串截取方案，结合代码示例解析技术要点。

## Unicode安全处理方案

通过`[]rune`转换实现字符级安全截取：

```go
func main() {
    str := "Hello, 世界!"
    runes := []rune(str)
    
    // 字符级安全截取
    sub3 := string(runes[7:9])
    fmt.Println("安全截取中文:", sub3) // 输出: 世界
}
```
